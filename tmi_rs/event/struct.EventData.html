pub struct EventData<T, Inner> where
    T: StringRef,
    Inner: Debug + Clone + Eq,  { /* fields omitted */ }

Content of a received message. Contains the sender, tags and and a generic Inner which contains the data specific to each event type.

Helper functions to access the contained data (message, target, specific tags) are implemented for for the EventContent<T, Inner> types they apply to.
