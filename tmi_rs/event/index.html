Modules

tags

Convenience methods to access the IRCv3 tags twitch sends. It's possible to access the tag values using their names with the MessageTags<T> trait. However, there are accessor methods for all documented tags which are only implemented for the event types where they are expected to be set. Additionally, some of the methods parse the tags and return more easily processed structs instead of the raw string values (see EmoteTag::emotes()), EmoteSetsTag::emote_sets()).
